#toPDF

toPDF es un proyecto de practica hecho con React, Javascript y Vite, en el cual el objetivo es lograr convertir un archivo con extensi√≥n .HEIC a .PDF.

Se agregaron las librerias "jsPDF" y "HEIC2any".
